## Research Code for Optimization and Learning

The Optimization and Machine Learning Lab hosts research and teaching code connected to optimization, machine learning, reinforcement learning, and computational methods for decision-making.

This organization collects earlier and companion projects from Martin Takac's research group, with emphasis on algorithms that can be inspected, reproduced, and extended.

<table>
  <tr>
    <td width="25%">
      <h3>Optimization</h3>
      <p>Methods for gradient-based learning, curvature, numerical algorithms, and efficient training.</p>
    </td>
    <td width="25%">
      <h3>Machine Learning</h3>
      <p>Models and experiments for representation learning, robustness, information bottlenecks, and data-driven prediction.</p>
    </td>
    <td width="25%">
      <h3>Reinforcement Learning</h3>
      <p>Algorithms and environments for sequential decision-making, games, and control under uncertainty.</p>
    </td>
    <td width="25%">
      <h3>Applications</h3>
      <p>Research code spanning energy systems, scheduling, EEG analysis, and scientific computing workflows.</p>
    </td>
  </tr>

## Public Repositories

| Repository | Focus |
| --- | --- |
| [Job-Shop](https://github.com/Optimization-and-Machine-Learning-Lab/Job-Shop) | Job shop scheduling experiments and notebooks. |

- Reinforcement learning for games, planning, control, and energy systems
- Scientific and engineering applications of learning methods
